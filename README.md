# Virat-Kohli-Dashboard
1.  Project Title / Headline
   Interactive Career Analytics Dashboard — Virat Kohli Performance Insights

2.  Short Description 
    An interactive data visualization dashboard that analyzes Virat Kohli’s international cricket career across formats and years. It provides key performance metrics such as total runs, average, strike rate, and scoring trends. Users can dynamically filter data by year range and match format to explore performance patterns. The dashboard transforms raw cricket statistics into clear, actionable visual insights.

3.  Tech Stack
   • 📊 Power BI Desktop – Main data visualization platform used for report creation.
   • 📂 Power Query – Data transformation and cleaning layer for reshaping and preparing the data.
   • 📁 File Format – .pbix for development and .png for dashboard previews.

4.  Data Source
    source: cricmetric.com

5.  Key Features 
   • 📊 Comprehensive Career Overview — Displays total innings, runs, batting average, and strike rate at a glance.
   • 🎯 Format-Based Filtering — Toggle between ODI, Test, and T20 formats for focused analysis.
   • 📅 Dynamic Year Range Selector — Interactive slider to analyze performance across different time periods.
   • 📈 Runs Trend Visualization — Year-wise bar chart to identify peak performance seasons.
   • 🧮 Milestone Tracking — Visual breakdown of centuries and half-centuries over time.
   • 🟣 Boundary Distribution Analysis — Pie chart showing proportion of fours vs sixes.
   • 🎨 User-Friendly Layout — Clear visual hierarchy with intuitive design for quick insights.
